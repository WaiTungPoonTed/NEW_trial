Some comman block tags

hr

The<hr> elemnt used to stand for "horizontal line"

address

The <address> elements indicates that the text is an address. For example,

Basecamp Potsdam, Kossätenweg 25
14476 Potdam-Golm
Germnay

blockquote

The <blockquote> element is used for extended quotations.

"The only thing i know is i know nothing". Socrates

details

The <details> element is used in conjunction with the <summary> tag.

The summaryThe rest of the details content goes here. If only <details> element is not supported, then the summary and the content are always visible.

Inline elements

Citing a source - cite

The <cite> makes it much easier for people and bots to search the code for this specific information. A great reference for these and other elements is "https://developer.mozilla.org/en-US/docs/Web/HTML"(with only <cite>)

(With <a href ="...">... <a>) https://developer.mozilla.org/en-US/docs/Web/HTML

Defining your abbreviations -abbr

The <abbr> will display an abbreviation on the browser. But if a title attribute is specified in the tag, the title will appear when the mouse hovers over the abbreviation. Screen readers also provide the title information.

The theory of general relativity is based on DG.

Specifying the use of time - time

The <time> is again a completely semantic tag that can be used to search for specific information.

The party starts at 9:00pm.

The party took place on New Year's Eve.

Identifying code fragments - code

The <code> element tends to be displayed in a different font, but also has the semantic meaning to convey information to as many people as possible.

import numpy as np
print(np.sin(np.pi))

Using subscripts and superscripts - sub and sup

The <sub> and <sup> elements are used to create the super and subscripts, e.g.

(x+y)2 = x2+2xy +y2, or H2O.
